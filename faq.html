Instant search

A collection of frequently asked questions and their answers.

  • How do I install LiquidBounce?

    Installing LiquidBounce is very easy. Just follow the steps described in the installation instructions.

  • Is LiquidBounce safe to use?

    Yes, it is safe to use LiquidBounce. We can guarantee that it does not contain any viruses, adware, crypto miners or other junkware. Also, LiquidBounce is open source. That means the source code is freely available. So if we ever violate our own statutes, it would be noticed immediately.

  • How do you make money if LiquidBounce is free?

    LiquidBounce is completely free. Nevertheless, we do have some ways to earn money with our work. On the one hand there are banners on this page, so if you want to support us for free, deactivate your AdBlocker, on the other hand we accept donations.

  • How do I update LiquidBounce?

    LiquidBounce will inform you after launch if a new version has been released. To update, simply download LiquidBounce again and overwrite the old file in your Mods folder. The next time you start LiquidBounce, you should have the latest version installed.

  • How do I change keybinds?

    Currently it is not possible to configure keybinds in the ClickGUI. Instead you have to use the bind command in chat. The syntax of the command is '.bind <module> <key>'; to remove a keybind, use '.bind <module> none'.

    Example: '.bind KillAura R'

  • My HUD is gone! How do I re-enable it?

    You might accidentally disable HUD. However, this problem can be easily addressed: Just open the ClickGUI by pressing RSHIFT and enable the HUD module which can be found in the render panel.

  • How can I make the ClickGUI smaller?

    If you have a small screen, the ClickGUI may be so large that you cannot see all the settings of certain modules. To solve this problem, right-click the module called ClickGUI and lower the scale value.

  • My KillAura isn't working! How can I fix it?

    There are many possible explanations for this problem. Before you contact us, check if one of the following approaches solves it:
    • Make sure players are marked as targets.
    • If AntiBot is enabled, disable it for testing.
    • Make sure you are not using mods which are known to cause such problems (e.g. BetterAnimations).
    • Some servers attempt to confuse KillAuras by pretending players are dead. To ensure that KillAura still attacks, activate Dead as target.

  • Is LiquidBounce compatible with all Forge mods?

    No, there are some mods that are not compatible with LiquidBounce (e.g. BetterAnimations). As a rule of thumb you can remember that mods that make major changes to Minecraft or its GUI will not work with LiquidBounce. So don't even try to use the client with large mod packs.
